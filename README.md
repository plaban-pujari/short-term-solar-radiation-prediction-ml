# Solar Radiation Prediction using Machine Learning 

##  Overview
This project predicts solar radiation using machine learning models based on weather parameters.

##  Features
- Data preprocessing and cleaning
- Machine learning model implementation
- Prediction and evaluation

##  Technologies Used
- Python
- Pandas
- Scikit-learn
- Matplotlib / Seaborn
- Streamlit (if used)

##  Results
The model successfully predicts solar radiation trends with reasonable accuracy.
